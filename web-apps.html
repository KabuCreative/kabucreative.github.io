Bespoke web applications

With our expertise in both web design and development, we are perfectly placed to help you build and launch a succesful web application.

We pride ourselves on:

  • Working with you to understand your objectives and market
  • Helping you prioritise your feature set - robustly debating the value of each feature

  • Providing training and support to help you manage your application

Case Study: BPCLEtool

Working with Darcy Associates, Python Charmers and the Victorian Department of Health, we helped to design and develop a web application to be used by hundreds of health services throughout Victoria.

BPCLEtool

The BPCLEtool concept was developed by Darcy Associates to help Victorian health organisations rate themselves against the BPCLE framework.

We helped transform a Microsoft Excel prototype of the tool into a feature-rich modern HTML5 web application.

Designing the User Experience

We worked with Darcy Associates to understand the application's requirements and design an optimal user experience.

Wireframing and user acceptance tests allowed us to validate key design concepts throughout the project.

The result was a clean and simple user interface and aesthetic design.

Build Process

Working as part of the Python Charmers team, our role included project management, design and backend development.

Core technologies used to build the application include the Django web framework, Javascript, jQuery and Bootstrap.

Working as part of the Python Charmers team, our role included:

  • Consulting with Darcy Associates to determine the optimal way to develop the required features
  • Managing remote members of the team responsible for different aspects of the project
  • User interface and web design
  • Back end web development in Django
  • Assisting in the roll out of the tool

User feedback has been overwhelmingly positive, and the tool has now been extended to become available to health services nation wide.
